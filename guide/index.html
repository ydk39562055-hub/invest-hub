로테이션 · 주도주 · 성장가속 — 각각 무엇을 대상으로, 어떤 규칙으로 종목을 뽑고, 등급을 어떻게 매기며, 무엇을 믿으면 안 되는지를 코드 기준으로 정리한 설명서입니다. 셋 다 "사라"가 아니라 볼 것을 좁혀주는 도구예요.

1

로테이션 랩

THEME · SECTOR ROTATION

지금 자금이 어느 테마·섹터로 몰리는지 상대강도 순위로 줄 세운다. "무엇을 살까"가 아니라 "어디가 뜨거운가"를 본다.

🎯대상

  • 미국 11개 섹터 + 한국 10개 섹터 + 내가 직접 짠 18개 테마(AI 인프라·전력/원자력 등)
  • 가격 데이터는 야후 파이낸스에서 매일 자동 수집 (미국 전일 종가)

📐어떻게 순위를 매기나

핵심은 상대강도(RS) — 벤치마크(지수)보다 얼마나 더 올랐나예요. 세 기간을 재서 섞습니다.

종합점수 = 0.5×(63일 상대강도) + 0.3×(21일) + 0.2×(5일)
  • 63·21·5 거래일 = 약 3개월·1개월·1주. 장기에 가장 큰 가중치를 준다(0.5).
  • 양수 = 지수를 이기는 중, 음수 = 지수에 지는 중. 이 점수가 높은 순으로 순위.
  • 테마 점수는 구성종목 상대강도의 중앙값(평균 아님) — 한두 종목 폭등에 순위가 왜곡되는 걸 막는다.

🧭같이 보는 보조 지표

지표뜻
확산도(breadth)그 테마에서 52주 신고가 –15% 이내에 든 종목 비율. 높을수록 몇 개가 아니라 다 같이 강한 것 = 건강한 강세.
거래대금 흐름최근 5일 평균 거래대금 ÷ 60일 평균. ×1.2 이상이면 자금 유입 가속.
🔄 로테이션 후보장기(63일)는 아직 하위 50%인데 단기(5일)가 상위 20% → 막 돌기 시작한 초입 후보.
RRG 4분면주도(우상)·차익(우하)·회피(좌하)·진입후보(좌상). 강한지 + 더 강해지는 중인지를 한 점으로.

🔁운영 주기

  • 매일 오전 8시 자동 실행 → 순위 계산 → 웹(깃허브 페이지) 자동 갱신 + 디스코드 알림
  • 3회 연속 수집 실패 종목은 상장폐지 의심으로 경고를 띄운다(계산에선 자동 제외).
2

주도주 레이더

LEADER RADAR

개별 주식에서 "이미 강한 대장"과 "바닥에서 막 도는 역발상"을 두 트랙으로 분리해 잡는다. (미국·한국 각각)

🚀트랙 A — 주도주 (이미 강한 대장)

통과 조건 — 상대강도 20일 지속 강세  +  52주 고점 대비 –25% 이내(고점 근처)  +  200일선 위·상승

강세장의 진짜 대장을 찾는 쪽. 이미 신고가 부근에서 힘이 유지되는 종목.

🔄트랙 B — 턴어라운드 (바닥 반등)

통과 조건 — 상대강도 변곡(바닥에서 반등 시작)  +  150일선 위·기울기 0 이상  +  고점 대비 –30%보다 더 빠진 종목

역발상 쪽. 많이 빠졌다가 도는 것이라 펀더멘털(실적) 검증 필수. 두 트랙은 성격이 반대라 섞으면 안 된다.

🏅금·은·동 등급 = 섹터 안에서의 서열

통과한 종목을 같은 섹터 안에서의 강도로 다시 줄 세운 것. 금속이 아니라 올림픽 메달 개념(1·2·3군).

금 · 골드섹터 내 최강 — 그 업종 진짜 대장
은 · 실버섹터 내 2군 — 강하지만 대장 아래
동 · 브론즈통과 종목 중 상대적 약체

📊같이 보는 것

  • 성격: 최근 10일 거래량이 50일 평균의 1.5배 이상 분출이면 🚀모멘텀, 아니면 😴조용.
  • 장세 게이트: 공격 장세 / 방어 장세를 판정 → 하락장이면 현금 대기를 권한다.
  • 7·20·60일 칸은 신호 발송 뒤 실제 수익률이 시간이 지나며 채워진다(미래를 미리 안 봄).

성장가속 신호

REVENUE ACCELERATION

미국 S&P500에서 매출 성장률이 빨라지는 종목을 다룬다. 단, 과거 검증 결과 때문에 공격이 아니라 방어(거부권) 중심으로 설계됐다.

🧪먼저 — 과거 검증이 시스템 모양을 정했다

부결매출 가속만 보고 그냥 사기(롱온리)는 지수를 못 이겼다. 평균은 +지만 소수 대박에 의존하고, 초과수익은 실적 발표 전에 이미 선반영돼 버린다.
검증됨밸류에이션 거부권 — 너무 비싼(주가매출배수 상위) 가속주는 초과수익이 0이었다. 즉 "비싼 건 빼기"가 유일하게 검증된 규칙.

그래서 이 시스템의 중심은 방어(거부권) + 선행지표 + 내 판단이다. "가속 종목을 다 사라"가 아님.

⚙️운영 순서 (한 사이클)

단계하는 일
① 가속 스크리너지금 매출이 가속 중인 종목을 후보로 추림(발굴).
② 밸류 거부권 ★너무 비싸면 🔴 STOP. 제일 먼저·제일 빠르게 버린다. (유일하게 검증된 신호)
③ 회계 red flag감사인 교체·재무제표 재작성·제출 지연·매출채권 급증이면 🔴 판단 보류.
④ 선행지표수주잔고·잔여계약(밀린 주문) 문장을 SEC 원문에서 후보로 뽑아 링크와 함께 제시.
⑤ 내 판단 + 기록물량이냐 단가냐 / 구조적이냐 사이클이냐를 사람이 분류하고, "이게 보이면 내가 틀린 것"을 적어둠.
⑥ 90일 복기시스템이 90일 뒤 채점을 리마인드.
실제 신호 = 가속 + 저평가 + 회계 깨끗의 교집합만.

🚦카드 색깔 읽는 법

🟢 초록자동 계산이 깨끗함 — 그대로 믿어도 됨(주가·매출·주식수 등).
🟡 노랑사람이 확인해야 함. 근사치거나 본문에서 긁은 문장 → 🔗 원문 대조 필수.
🔴 빨강거부권 발동 또는 감사 실패 — 판단 보류.
과신 금지. 검증된 건 밸류 거부권 하나뿐이다. 가속·선행지표는 "보조 재료"이고, 회계 거부권은 정형 신호(항목코드·지연·매출채권)만 잡는다 — 공매도 리포트·규제 조사 같은 비정형 의혹은 못 잡으니 사람이 따로 챙긴다. 카드는 "매수 리스트"가 아니라 "공부 리스트".
